# dotfiles

My personal configuration files for macOS.

## Initial install

Simply run the following command to copy all files in your home directory.

```
source bootstrapp.sh
```

## Update zshrc

Add this entry to your `.zshrc` file.

```
source ".zshrc_base"
```
